# Urgence Marseilla Bot

Bot Discord pour le serveur **FiveM Urgence Marseilla**.

## Fonctionnalités

- Gestion automatique des rôles Staff et Urgence
- Création des catégories et salons (textuels et vocaux)
- Salons vocaux BDA numérotés et vocaux pour chaque rôle Staff et Urgence
- Commande whitelist pour FiveM
- Commande setup pour tout configurer automatiquement

## Installation

```bash
git clone https://github.com/ton-utilisateur/urgence-marseilla-bot.git
cd urgence-marseilla-bot
npm install
